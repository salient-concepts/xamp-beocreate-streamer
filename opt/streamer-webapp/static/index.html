About xAMP

xAMP Audio Streamer is a custom hi-fi streaming OS built on top of Raspberry Pi OS. It runs MPD as the audio engine, exposes a branded web UI for control (this app), and ships with AirPlay 2 support so any iPhone or Mac can stream to it natively.

The hardware on this unit is the —, a 4-channel SigmaStudio DSP DAC built around the ADAU1452 + PCM5122 + dual TAS5754M class-D amplifiers. Designed for Bang & Olufsen Beocreate active-speaker conversions, it provides 2x30W + 2x60W of amplification with parametric EQ, crossovers, and time-alignment in firmware — all driven by the Pi over I²S.

Features included: 50,000+ internet radio stations via radio-browser.info, curated featured list of audiophile favourites (Radio Paradise FLAC, Linn, Naim, BBC R3, SomaFM, Jazz24, ABC Classic, etc.), favourites star button, hi-res FLAC playback up to 24-bit / 192 kHz, real-time spectrum visualizer in the web app, AirPlay 2 receiver, and five colour themes.

Audiophile Customization
This unit ships with a working baseline. The Beocreate platform is designed to be tuned to your specific speakers and listening preferences — here is what is fixed vs. yours to customize.

What is configured (fixed working baseline):

  • HiFiBerry’s canonical 4way-default.xml DSP profile is loaded into the Beocreate EEPROM and auto-loaded at every boot via SELFBOOT
  • IIR filter slots cleared to flat passthrough on each boot (boot service: dsp-clear-filters.service)
  • Pi-side audiophile tuning: performance CPU governor, MPD pinned to dedicated isolated core 3, MPD nice=-15, RT scheduling enabled, HDMI audio disabled, USB audio class blacklisted, WiFi power-save off
  • MPD ALSA buffer 16384, no software resampling — native bit-perfect path from source to DAC
  • Output Voltage Limiter jumper removed (HiFiBerry recommendation for their DSP profiles)
  • SELFBOOT jumper installed; MUTE jumper not installed

What is yours to customize:

  • Speaker preset / crossover / EQ — the default is flat passthrough. Use HiFiBerry’s free BeoCreate2 web app (a NodeJS application) to select a B&O speaker preset, configure 2-/3-/4-way crossovers, apply parametric EQ, and time-align drivers. sigmatcpserver is pre-installed on this unit and listens on port 8086 for SigmaStudio / BeoCreate2 to connect.
  • PCM5122 digital filter mode — currently at silicon defaults. Linear-phase, minimum-phase, apodizing, brick-wall, and slow-roll-off variants are selectable via SigmaStudio (Windows, free from Analog Devices) by editing the DSP profile and re-flashing with dsptoolkit install-profile.
  • Custom DSP profiles — swap the EEPROM-loaded .xml at any time. The boot service auto-applies whatever is at /var/lib/hifiberry/dspprogram.xml.
  • Source software — the alternatives section below lists Volumio, Moode, RoonBridge, etc. Flash any of them to the SD card and the unit becomes that streamer.

A Salient Concepts small-batch audiophile build. Custom hardware paired with a fully open-source software baseline, designed for hands-on owners comfortable with command-line and DSP tooling — ready to tune to your specific speakers and listening room. Each unit ships as a complete working baseline; ongoing customization draws on the broader open-source ecosystem (HiFiBerry, BeoCreate2, MPD, shairport-sync).

Alternative Software
Not locked in. The buyer can flash any of these to the SD card and the unit becomes a different streamer.
Volumio free / €30
Most popular Pi audiophile OS
Mature, broad codec/source support, polished web UI, paid Premium tier adds DSP and multi-room. Wipes xAMP — flash the SD card with Volumio's official image.
volumio.com
Moode Audio free
Pure audiophile-focused, open source
Deep DSP options, parametric EQ, room correction, integrated Camilla DSP. No paid tier, no telemetry. Hi-res FLAC, MQA, DSD support. Flash to SD.
moodeaudio.org
piCorePlayer + LMS free
Squeezebox / Lyrion ecosystem
Tiny footprint Squeezelite endpoint. Pairs with Lyrion Music Server (formerly Logitech Media Server) running on a NAS or PC. Multi-room sync across all your Squeezebox devices.
picoreplayer.org · lyrion.org
RoPieee (Roon) $130/yr
Best-in-class library + metadata
Turns the Pi into a Roon Ready endpoint. Requires Roon Core running on a separate NAS or PC ($130/year subscription or $830 lifetime). Industry-best metadata and library management.
ropieee.org · roonlabs.com
HQPlayer NAA €230
Audiophile upsampling holy grail
Pi runs as a thin Network Audio Adapter for Signalyst's HQPlayer Embedded. Pairs with the most respected upsampler in computer audio. Best DSD512 and PCM upsampling available.
signalyst.com
DietPi + MPD free
Roll your own minimal stack
Lightweight Debian-based OS with one-click MPD installer. For users who want to build their own configuration from scratch. Same MPD that xAMP uses internally.
dietpi.com
How to switch: power down the unit, remove the SD card, flash the new OS image with Raspberry Pi Imager or Balena Etcher, reinsert and power on. Most options auto-detect the HiFiBerry Beocreate DAC (use the hifiberry-dac overlay). Switching wipes xAMP and the favourites database — keep a backup of your SD card if you want to come back.
